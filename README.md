# Davis Portfolio

Problem-first portfolio site designed for fast scanning by recruiters and hiring managers.

## Quick Summary

- Uses a Problem -> Solution -> Impact structure across key sections.
- Highlights quantified outcomes: latency, throughput, growth, and time saved.
- Includes credibility signals: awards, compliance alignment, benchmarks, and validation method.
- Uses reusable UI primitives (`Hero`, `ValidationTicker`, `GlassCard`) for maintainability.
- Keeps assets centralized in `public/imgs/`.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Project Standards

- Portfolio layout patterns: `PORTFOLIO_PATTERNS.md`
- Project showcase/readme spec: `PROJECT_SPEC.md`
